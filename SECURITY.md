# Security Policy

Harness Kernel is pre-1.0. Please report security issues privately when
possible.

## Reporting

Email security reports to duckssoftwarecontact@gmail.com.

You may also use GitHub private vulnerability reporting for this repository if
it is enabled. If neither private reporting nor email is available, open a
minimal public issue asking for a security contact and do not include exploit
details.

Please include:

- affected package and version or commit;
- impact and affected runtime surface;
- reproduction steps or proof of concept;
- any known mitigation.

## Supported Versions

Security fixes target the latest released `0.x` line unless a release note says
otherwise.
