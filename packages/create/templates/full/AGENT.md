# Agent Project Notes

This is a generated full Harness Kernel project.

- Edit `src/agent.ts` to compose modes, hooks, and roles.
- Edit `src/modes/` to change prompts, tools, and active context providers.
- Edit `src/context/` to add context providers.
- Edit `src/events/` to add typed event classes.
- Edit `src/hooks/` to react to harness events.
- Edit `src/tools/` to add domain-specific tools.
- Edit `src/run.ts` to change runner behavior, default model wiring, logging,
  approval, storage, sandbox, or streaming output.

Runtime model providers, session model selection, credentials, sandbox, and run
storage belong to the runner side of this project, not the agent definition itself.
