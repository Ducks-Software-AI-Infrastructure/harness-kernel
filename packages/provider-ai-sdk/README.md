# @harness-kernel/provider-ai-sdk

Vercel AI SDK model provider integration for Harness Kernel.

```sh
pnpm add @harness-kernel/provider-ai-sdk ai
```

The application owns model provider configuration and credentials, then passes
providers into `createHarnessSessionStore`.
